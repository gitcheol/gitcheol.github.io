# Vim editor 

#### vim의 4가지 모드 

1. 편집 모드 
2. 명령 모드
3. Ex명령 모드
4. Visual 모드 

#### 편집모드

```python
#enter the input mode 
'a' or 'i' : 편집모드로
O : 커서 아래에서 new line만들고 편집모드로
o : 커서 위로 new line만들고 편집모드로
I : 라인 제일 앞으로 이동 후 편집모드로
A : 라인 제일 뒤로 이동 후 편집모드로
 
  
#편집, 명령모드 명령어
shifh + 방향키 : 한 단어씩 이동 
```



### 명령모드

```python
shift 4 : 커서 라인 맨 뒤로 이동
shift 6 : 커서 라인 맨 앞으로 이동 
shift 3 : 커서에 있는 단어 찾기 
gg : 첫 line 이동
G : 마지막 line 이동 
x : 현재 커서 위치 문자 삭제
dw : 단어삭제 
yy : 행 복사 # 3yy = 현재 커서부터 3행 복사 
dd : 행 잘라내기 # 3dd = 현재 커서부터 3행 잘라내기 
p : 복사 or 잘라낸 행 붙여넣기
u : 되돌리기
ctrl + r : 되돌리기 취소
/문자열 : 문자열 찾기 
r : 커서 글자 변경 
  
'''tip'''
#대/중괄호 빨리찾기
{}기호에서 %기호 누르면 해당 중괄호 시작 or 끝으로 이동 
#문자열 빨리 찾기 
문자열에 커서 두고 * or # 누르면 문자열 찾기랑 같으 n or N으로 이동 
#함수 or 지역변호 선언부로 바로 이동 
함수 시작부분 : [i 
#탭 삽입 
숫자>> : 현재 커서부터 라인만큼 tab이동 

   
```



#### ex명령모드(:)

```python
w,wq,q,q!: 파일 저장 및 나가기 
w file.txt : file.txt로 파일 저장 
e filf.txt : file.txt파일 불러옴
e# : 바로 이전에 열었던 파일을 불러옴
e : 현재 파일을 불러옴
  
단어 바꾸기 : %s/before/after/g #before -> after
  # g: 매칭되는 문자열을 물어보지 않고 변경
  # i: 대소문자를 구분하지 않고 변경 (ignore cases)
  # c: 매칭되는 문자열마다 바꿀 것인지 사용자에게 물어보고 변경
  
  
숫자 : 해당 line으로 이동 
sp : 화면 가로분할 #q로 분할 해제 
vs : 화면 세로분할 #q로 분할 해제 
```



#### Visual모드

```shell
'블록 설정' + '=' #source코드 정렬(c, c++, java한정 (){}있는 언어)
'블록 설정' + '>>' # tab추가 
'블록 설정' + 'y' #블록 복사 p로 불텨넣기 
U : 대문자로 바꾸기
u : 소문자로 바꾸기
~ : 대소문자 전황
d : 지정 블록 지우기 
```
